# Documentation de recette web

Ce dossier regroupe les documents de validation fonctionnelle du front web.

Contenu :

- `recette-web.md` : plan de recette et périmètre du front web.
- `jeux-de-tests-web.md` : jeux de données et profils de test.
- `cahier-de-tests-web.md` : cas de tests détaillés couvrant les fonctionnalités réellement présentes.
- `script-de-test-web.md` : déroulé recommandé pour exécuter la recette web.

Périmètre web actuellement couvert :

- page de connexion
- page d'inscription
- protection des routes privées
- layout d'administration
- tableau de bord
- navigation vers pages placeholders
- déconnexion

Le dossier est volontairement aligné sur l'état réel du projet `front-web` afin de garder une recette crédible et exploitable.
